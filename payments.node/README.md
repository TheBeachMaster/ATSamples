# <u>Sample USSD applications with Payments and SMS</u> 

### Running this sample 

1. ```bash
   $ npm i --save
   ```


   ```bash
   $ npm start
   ```

   

2. ```powershell
   ngrok http 3008
   ```

   or

   ```bash
   ngrok 3008
   ```

3. Your callback urls are `~/pay` for payments and `~/ussd` for ussd 

4. In the `config.js` , add Africa's Talking credentials
